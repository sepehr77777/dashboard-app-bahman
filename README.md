# داشبورد ادمین - Next.js + TypeScript

این پروژه یک داشبورد مدیریتی است که با استفاده از Next.js و TypeScript پیاده‌سازی شده است.

## ویژگی‌ها

- ورود (Login) با استفاده از DummyJSON API
- محافظت از مسیر داشبورد (Protected Route)
- نمایش آمار کاربران
- نمایش آمار محصولات
- اتصال به API با Axios
- معماری مبتنی بر App Router

## تکنولوژی‌های استفاده شده

- Next.js (App Router)
- TypeScript
- Axios
- React Hooks
